<Section icon={Swords} title="กลไกการต่อสู้และรูปแบบการเล่น">

การเชี่ยวชาญตาราง 3x3 คือกุญแจสำคัญในการเอาชีวิตรอดจากเหล่าฝันร้าย (Nightmares) การต่อสู้เป็นแบบเรียลไทม์ ซึ่งหมายความว่าจังหวะการลงตัวละครของคุณมีความสำคัญไม่แพ้ค่าสถานะของตัวละคร

**ไฮไลท์วิดีโอ:**
- ทำความเข้าใจค่า Deployment Cost ในการต่อสู้แบบเรียลไทม์
- การปรับเปลี่ยนรูปแบบกลยุทธ์เพื่อรับมือกับบอส
- การใช้ Ultimate Skills เพื่อสร้างผลลัพธ์สูงสุด

<YouTubeEmbed videoId="_0GgrGHsHb8" title="วิดีโอการเล่น Illusion Connect: Re ช่วงเปิดตัว" />

ระบบการต่อสู้เน้นไปที่การจัดการทรัพยากรและการแพ้ทางของคลาส ใช้ข้อมูลด้านล่างนี้เพื่อเพิ่มประสิทธิภาพในการต่อสู้ของคุณ:

| กลไก | ผลกระทบต่อการต่อสู้ |
| :--- | :--- |
| **Deployment Cost** | กำหนดความเร็วในการลงตัวละคร Radiants |
| **ตาราง 3x3** | ช่วยให้วางกลยุทธ์ตามเลนและการปกป้อง |
| **Ultimate Skills** | ท่าไม้ตายพลังทำลายล้างสูงเพื่อเคลียร์ศัตรูหรือทำลายเกราะ |
| **การแพ้ทางคลาส** | จำเป็นสำหรับการลดความเสียหายจากเหล่า Nightmares |

<Callout type="tip">
อย่ารีบลงตัวละคร Radiants ทั้งหมดพร้อมกัน ให้เก็บตัวละครที่ใช้คอสต์สูงไว้จนกว่าคุณจะตั้งแนวหน้าได้สำเร็จ เพราะการวางจังหวะที่ผิดพลาดมักจะทำให้ Leader ของคุณได้รับความเสียหายโดยไม่จำเป็น
</Callout>

<Section icon={BookOpen} title="กลยุทธ์การ Reroll และการพัฒนา">

การ Reroll ช่วยให้คุณเริ่มต้นเกมด้วยความได้เปรียบอย่างมาก ทำตามขั้นตอนเหล่านี้เพื่อให้แน่ใจว่าบัญชีของคุณได้รับการปรับให้เหมาะสมตั้งแต่วันแรก

<Steps>
  <Step num="1" title="บัญชีเริ่มต้น">
    เข้าสู่ระบบในฐานะ Guest เพื่อทำการสุ่มกาชาเริ่มต้นโดยไม่ต้องผูกบัญชีถาวร
  </Step>
  <Step num="2" title="ทำบทช่วยสอน">
    เล่นบทช่วยสอนเริ่มต้นให้จบเพื่อปลดล็อกเมนูการสุ่มและรับรางวัลทั้งหมดจากกล่องจดหมาย
  </Step>
  <Step num="3" title="การสุ่มกาชา">
    ใช้ตั๋วฟรีกับตู้ Eternal Summon จากนั้นใช้เพชรกับตู้ Selected UP เพื่อหาตัวละคร SSR เป้าหมายของคุณ
  </Step>
  <Step num="4" title="การผูกบัญชี">
    เมื่อคุณสุ่มได้ตัวละคร SSR เก่งๆ 2-3 ตัว (เช่น Yuffie หรือ Anna) ให้ทำการผูกบัญชีเพื่อบันทึกความคืบหน้า
  </Step>
</Steps>

</Section>

<Section icon={Shield} title="アカウント管理と削除の警告">

パワーユーザーにとって最もフラストレーションが溜まる側面の一つが、アカウント管理システムです。2026年現在、『イリュージョンコネクト：Re』では、削除対象としてマークされたアカウントに対して30日間の「冷却期間」が厳格に適用されます。これは、特定のサーバーにおける連携アカウントとゲストアカウントの両方に適用されます。

リセマラ中にこれを回避するには、「アカウント削除」機能ではなく「アカウント切り替え」機能を使用する必要があります。

1.  **アカウント切り替え**: ゲーム設定内にあります。ログイン画面に戻ることができます。
2.  **サーバーホッピング**: ゲストの場合は、単純に別のサーバーを選択します（例：サーバー1からサーバー5へ移動）。
3.  **メールのエイリアス**: メールアドレスが `example@gmail.com` の場合、`example+1@gmail.com` として登録できます。ゲーム側は新しいメールアドレスとして認識しますが、すべての確認コードはメインの受信トレイに届きます。

| 方法 | 削除時間 | 適した用途 |
| :--- | :--- | :--- |
| **標準削除** | 30日 | 恒久的な削除 |
| **サーバー交換** | 即時 | クイックリセマラ |
| **新規メール** | 即時 | 同一サーバーでの新規開始 |

<Callout type="warning" title="連携の警告">
SSRの引きに100%満足するまで、アカウントをメインのSNSやメールに連携しないでください。一度連携すると、30日間のサイクルを待たずに連携を解除するのは非常に困難です。
</Callout>

</Section>

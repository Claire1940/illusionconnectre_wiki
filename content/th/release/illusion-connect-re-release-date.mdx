การสร้างทีมที่มั่นคงมีความสำคัญมากกว่าการไล่ตามตัวละครหายากที่ขาดความเข้ากัน (Synergy) ทีมที่สมดุลควรให้ความสำคัญกับการครอบคลุมบทบาท: แทงค์ 1 ตัวเพื่อรับดาเมจแนวหน้า, ฮีลเลอร์ 1 ตัวเพื่อประคองทีม, และตัวทำดาเมจ 2 ตัวเพื่อเคลียร์เวฟศัตรู

ตารางด้านล่างแสดงการกระจายบทบาทที่แนะนำสำหรับการเล่นช่วงต้นเกม:

| บทบาท | หน้าที่ | ลำดับความสำคัญ |
| :--- | :--- | :--- |
| **Guardian** | รับดาเมจและปกป้องทีม | สูง |
| **Sorcerer** | ดาเมจ AoE และเคลียร์เวฟศัตรู | สูง |
| **Healer** | ประคองทีมและฟื้นฟูเลือด | ปานกลาง |
| **Attack** | ดาเมจเป้าหมายเดี่ยวสำหรับบอส | ปานกลาง |
| **Support** | บัฟและควบคุมรูปแบบการต่อสู้ | ต่ำ |

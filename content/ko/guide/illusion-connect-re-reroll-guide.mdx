**일루전 커넥트: Re 리세마라 가이드**에서 가장 큰 걸림돌 중 하나는 30일 계정 삭제 정책입니다. 게스트 계정을 삭제하려 하면 데이터가 한 달 동안 유지됩니다. 이를 우회하려면 다음 전문적인 전략을 사용하세요.

### 다중 서버 방법
게임은 현재 12개 이상의 서버를 운영 중입니다. 각 서버는 게스트 계정에 대해 별도의 프로필을 유지합니다.
1. **서버 선택** 메뉴를 엽니다.
2. 새 서버를 선택합니다 (예: 서버 1에서 서버 2로 이동).
3. 튜토리얼을 다시 시작합니다.
4. 이 방법을 사용하면 데이터 삭제나 이메일 변경 없이 12번의 리세마라가 가능합니다.

### 이메일 점(Salted) 방법
모든 서버를 다 사용했는데도 미유키를 얻지 못했다면, 계정 등록 시 '점(Salted)' 이메일을 사용하세요.
- 이메일이 `gamer@gmail.com`이라면 `gamer+1@gmail.com`으로 등록합니다.
- 게임은 이를 고유한 이메일로 인식하지만, 인증 코드는 모두 메인 편지함으로 옵니다.
- 이를 통해 원하는 서버에서 무한정 시도할 수 있습니다.

| 방법 | 장점 | 단점 |
| :--- | :--- | :--- |
| **서버 전환** | 즉시 가능, 새 이메일 불필요 | 먼 서버는 지연 시간 발생 |
| **이메일 점 방법** | 한 서버에서 무한 시도 | 매번 계정 등록 필요 |
| **데이터 삭제** | 앱 상태 초기화 | 느림, 데이터 재다운로드 필요 |

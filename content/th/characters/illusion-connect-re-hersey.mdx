- ควรส่ง **Radiants** ลงสนามโดยคำนึงถึง **การจัดการค่า Cost** และ **การแก้ทางคลาส** ในระหว่างการต่อสู้แบบเรียลไทม์

- การอัปเกรด **Dream Base** จะช่วยมอบโบนัสสถานะถาวรที่เพิ่มประสิทธิภาพการต่อสู้ของทีมได้อย่างมหาศาล

<Section icon={Swords} title="พื้นฐานกลยุทธ์การจัดทีม">

การสร้างทีมที่มีประสิทธิภาพใน Illusion Connect: Re ไม่ใช่แค่การเลือกตัวละครระดับสูงเท่านั้น แต่ความสำเร็จขึ้นอยู่กับการสร้างสมดุลของค่า Cost ในการลงสนาม, การเข้าใจความสัมพันธ์ของคลาส และการตอบสนองต่อสถานการณ์ในการต่อสู้แบบเรียลไทม์

**ไฮไลท์วิดีโอ:**
- การวางตำแหน่ง Radiants บนตาราง 3x3 อย่างมีกลยุทธ์
- การจัดการค่า Cost เพื่อจังหวะการใช้สกิลที่เหมาะสมที่สุด
- การระบุคลาสที่แก้ทางกันเพื่อทำลายรูปแบบทีมของ Nightmare

<YouTubeEmbed videoId="_0GgrGHsHb8" title="Illusion Connect: Re Official Launch Gameplay" />

ทีมที่ดีควรใช้กลยุทธ์ "Core Shell" เพื่อให้มั่นใจว่าคุณมีการครอบคลุมที่เพียงพอสำหรับรับมือกับการเผชิญหน้าในรูปแบบต่างๆ

| บทบาท | หน้าที่ | ลำดับความสำคัญ |
| :--- | :--- | :--- |
| **แนวหน้า (Frontline)** | รับความเสียหายและปกป้องแนวหลัง | **สูง** |
| **AoE DPS** | กำจัดศัตรูเป็นกลุ่มและเคลียร์มอนสเตอร์ | **สูง** |
| **Single Target** | สร้างความเสียหายแก่บอสและกำจัดศัตรูตัวฉกาจ | **ปานกลาง** |
| **ฮีล/ซัพพอร์ต (Sustain)** | รักษา HP ของทีมในการต่อสู้ระยะยาว | **สูง** |
| **สนับสนุน (Support)** | บัฟ, ดีบัฟ และควบคุมจังหวะการต่อสู้ | **ปานกลาง** |

<Callout type="tip" title="เคล็ดลับการประสานพลัง">
จับคู่ตัวทำดาเมจหลักของคุณกับตัวละครที่สามารถดีบัฟลดพลังป้องกันศัตรูได้เสมอ เพื่อเพิ่มความเสียหายสูงสุดในช่วงเวลาที่ใช้ Ultimate Skill
</Callout>

<Section icon={Target} title="การเพิ่มประสิทธิภาพทีม Radiant">

เมื่อจัดทีม ให้จัดลำดับความสำคัญของตัวละครที่ทำได้หลายบทบาทหรือมอบบัฟติดตัวที่แข็งแกร่งให้กับสมาชิกทีมที่เหลือ ตารางต่อไปนี้สรุปวิธีประเมินตัวละครที่คุณมีเพื่อให้ได้ประสิทธิภาพสูงสุด

| ระดับความหายาก | ลำดับการลงทุน | หมายเหตุประสิทธิภาพ |
| :--- | :--- | :--- |
| **SSR** | ลำดับแรกสำหรับตำแหน่งหลัก | ศักยภาพในการเติบโตสูง, ใช้ทรัพยากรสูง |
| **SR** | รองลงมาสำหรับสถานการณ์เฉพาะ | คุ้มค่า, ปลุกพลัง (Awaken) ได้ง่ายกว่า |
| **R** | สำหรับใช้งานเฉพาะทางเท่านั้น | ใช้อย่างจำกัดในช่วงต้นเกม |

  </Step>

  </Step>
